# 맥북 입문자 (Windows 유저) 를 위한 필수 적응 가이드

Windows 환경에서 Mac (M1 이상) 환경으로 넘어오신 분들을 위한 핵심 요약 가이드입니다.

## 1. 핵심: `Ctrl`의 역할은 이제 `Command(⌘)`가 합니다!
가장 헷갈리는 부분입니다. 윈도우에서 쓰던 단축키의 `Ctrl`을 `Command(⌘)`로 바꿔서 생각하세요.

| 기능 | Windows | Mac |
| :--- | :--- | :--- |
| **복사** | `Ctrl + C` | **`Cmd + C`** |
| **붙여넣기** | `Ctrl + V` | **`Cmd + V`** |
| **자르기** | `Ctrl + X` | **`Cmd + X`** |
| **저장** | `Ctrl + S` | **`Cmd + S`** |
| **새로고침** | `F5` | **`Cmd + R`** |
| **실행 취소** | `Ctrl + Z` | **`Cmd + Z`** |

<br/>

## 2. 한영 전환
* **가장 편한 방법 (기본값):** `Caps Lock` 키를 짧게 한 번 누릅니다. (길게 누르면 대문자 고정 모드가 됩니다.)
* **구형 맥 / 이전 방식:** `Cmd + Space` (현재 기본 설정으로는 Spotlight 검색창 띄우기로 설정된 경우가 많으나, 시스템 설정 -> 키보드에서 변경 가능합니다.)

<br/>

## 3. 개발자 핵심 팁: VS Code 중요 단축키
개발 효율을 높여주는 필수 단축키 모음입니다.

* **파일 이름으로 바로 찾기 (Go to File):** `Cmd + P` (원하는 파일명 입력 후 엔터)
* **전체 디렉토리 내 텍스트 검색 (Search in Files):** `Cmd + Shift + F`
* **코드 줄 상하로 이동시키기:** 수정할 줄에 커서를 두고 `Option(⌥) + 위/아래 방향키`
* **코드 줄 그대로 아래로 복사하기:** `Option(⌥) + Shift + 위/아래 방향키`
* **왼쪽 사이드바 열기/닫기:** `Cmd + B`
* **내장 터미널 열기/닫기:** `Ctrl + \`` (숫자 1 키보드 옆의 백틱 기호)

<br/>

## 4. 창과 앱 전환 (알트탭의 진화)
* **앱 (프로그램) 간 전환:** `Cmd + Tab` (예: 크롬 <-> VS Code)
* **같은 앱 내 창끼리 전환:** `Cmd + \`` (예: 크롬 창이 2개 이상 띄워져 있을 때 크롬 창들끼리만 전환)

<br/>

## 5. 화면 캡처 사용법
별도 캡처 프로그램 없이 내장 기능이 매우 강력합니다.

* **전체 화면 캡처:** `Cmd + Shift + 3`
* **✨부분 캡처 (가장 많이 씀):** `Cmd + Shift + 4` 누르고 원하는 영역 마우스 드래그
* **특정 창/메뉴만 캡처:** `Cmd + Shift + 4` 누르고 `Space bar` 누른 뒤 마우스가 카메라 모양으로 변하면 원하는 창 클릭
* **화면 녹화 및 캡처 옵션 메뉴 열기:** `Cmd + Shift + 5`

<br/>

---
**💡 팁:** 
처음 1~2주는 습관적으로 `Ctrl` 위치에 새끼손가락이 가서 어색하겠지만 (특히 터미널에서 서버 끌 때 `Ctrl+C`는 그대로 씁니다!), 엄지손가락으로 `Command`를 누르는 습관이 한 번 들면 윈도우 단축키보다 손가락 동선이 짧아져 피로도가 확 줄어드는 것을 느끼실 수 있습니다. 화이팅입니다! 🚀
