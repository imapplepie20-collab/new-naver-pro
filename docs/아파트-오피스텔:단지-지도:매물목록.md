# 단지 와 임시 매물 목록 과 정규 매물 목록
## 아파트 오피스텔
### 아파트
- 아파드 단지 목록 이나 아파트 단지 지도에서 단지를 클릭 하면 아파트 임시 매물 목록 페이지 이동
- 아파트 임시 매물 목록 페이지
    -- 해당 단지의 매물의 모든 정보를 그리드 형식으로 보여줌
        --- 개별 삭제
        --- 그룹 삭제
    -- 엑셀 다운로드
        --- 엑셀의 모든 필드 선택 출력 기능
    -- 저장 버튼
        --- 정규 매물로 저장 후 정규 매물 목록으로 이동 ----**
    -- 단지 목록 버튼(부동산 매물 검색)
        --- 단지목록으로 이동
    -- 아파트 정규 매물 목록 버튼
        --- 정규매물로 저장후 아파트 정규 매물 목록으로 이동 ----**

- 아피트 정규 매물 목록 페이지 ----** 이 데이타 중앙 디비에 저장 (사용자 아이디로 저장)
    -- 매물의 모든 정보를 그리드 형식으로 보여줌
        --- 개별 삭제
        --- 그룹 삭제
    -- 엑셀 다운로드
        --- 엑셀의 모든 필드 선택 출력 기능
    -- 단지 목록 버튼(부동산 매물 검색)
        --- 단지 목록 으로 이동
